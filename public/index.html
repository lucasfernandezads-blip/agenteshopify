Como Publicar este App 100% Online

Para colocar esse agente rodando 24h/7d na nuvem sem precisar deixar seu computador ligado, siga um dos métodos fáceis abaixo:

Opção 1: Render.com (Recomendado - Grátis/Barato)

  1. Envie esta pasta de código para um repositório no seu GitHub.
  2. Acesse o Render.com e crie um Web Service.
  3. Conecte seu repositório do GitHub. O Render detectará automaticamente o `Dockerfile` ou ambiente Node.js!
  4. Nas variáveis de ambiente no Render, insira suas chaves ou configure direto no Painel Web.

Opção 2: Railway.app

  1. Acesse Railway.app e clique em Deploy from GitHub repo.
  2. O Railway fará o build automático usando `npm run build` e `npm start`.
  3. Copie a URL gerada e cole no Webhook do BotConversa!
